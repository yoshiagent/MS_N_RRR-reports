🧰 元件工具箱 — P 系列互動元件 (P1, P2…)

這裡是 **P 系列互動元件** 目錄(掛到既有圖上的元件,如逐日回放)。每個元件有編號 P1 / P2…;你想加哪個,直接跟我說對應那句話(或說「在某圖加 P1」),我就照組裝法一行套上。新元件做好會自動出現在這頁。
編號系列:B 系列 = 可堆疊的「內容積木」(B1 CMG · B2 RR籌碼流向 · B3 雙K…,見 🧱 blocks);P 系列 = 掛到圖上的「互動元件」(下方)。P 再分兩次類:**互動行為**(跨圖通用 UI,如逐日回放)、**K 圖層 overlay**(掛到 K 圖的資料層,如成本線/SH·SL,共用模組 kchart_overlays,一行套到任意 K 圖)。
〘 互動行為 〙
P1　⏮ 逐日回放　✅ 可用　〔fn: chart_buttons.day_replay〕
在相位圖加「◀ 前一日 / 最新 / 後一日 ▶」按鈕,沿軌跡逐日回放,旁邊即時顯示當日日期 + 兩軸座標。
💬 想加 → 直接跟我說:「在 ○○ 相位圖加『逐日回放』(P1)」
一行用法day_replay(div_id, xs, ys, dates, bar_id="唯一前綴", x_label="X軸名", y_label="Y軸名")
需求該圖要有逐日歷史軌跡(xs/ys ≥ 2);圖的 marker 是最後加入的 plotly trace。
已用於B1 cmg_block(南電 CMG 相位圖)
P9　⏱ 範圍/平移/縮放鈕　✅ 可用　〔fn: chart_buttons.chart_toolbar_html + chart_js_helpers_block〕
時間序列圖範圍鈕。兩種版型:(A) 正典全功能 chart_js_helpers(近N日+平移+縮放+重設,chartRangeDays/chartPan/chartZoom/chartReset);(B) CIO 偏好精簡版=只『近3月/近6月/全部』、**三鈕右端都固定最後一天+右側留白**(切範圍時最後一天不動、末端標記不被切)、取消平移縮放。皆配 chart_locked_config+dragmode=False+chart_css(touch-action)=桌面只 click+雙擊、手機只 tap+長按。
💬 想加 → 直接跟我說:「在 ○○ 圖加『範圍鈕』(P9;說全功能或精簡右端固定版)」
一行用法精簡版: 預先算 RANGES={key:[lo, 最後一天+留白]} + rtRange(div,key) IIFE(見 rotation_timing_chart)
需求時間序列圖(date x)。全功能版頁面含 chart_js_helpers_block();注入頁用 range_buttons(自帶)。
已用於B4 換股時機圖(精簡右端固定版:近3月/近6月/全部)
P10　✛ 點擊定游標 + 按鈕移動　🟡 各圖自帶(待通用化)　〔fn: (rotation_timing _cursor_js / daily_weekly _JS)〕
點圖／手機點觸 = 定游標(垂直線);◀◀◀▶▶▶ 按鈕 + 鍵盤 ←→ 移動游標(圖不動、不隨滑鼠);游標處當日資料寫到圖上方資訊列。x 用數字索引 → 點擊像素線性換算成索引(穩健)。
💬 想加 → 直接跟我說:「在 ○○ 圖加『點擊定游標 + 按鈕移動』(P10)」
一行用法_cursor_js(div, dates, z, ...)(B4 範式);x 須數字索引、hovermode=False、doubleClick=False
需求x 軸數字索引(每單位1日);config doubleClick=False。
已用於B4 換股時機圖 · B3 daily_weekly(各自實作);待第 3 消費者抽共用 JS
