📐 CMG 公式說明 — NBC15_Z60 / SBD_Z60

🧱 B1 · 📖 說明
CMG 相位圖兩軸 NBC15_Z60(X)、SBD_Z60(Y) 都是「某個日頻分點籌碼指標的 60 日 Z-score」。下方列出每日原始值 → Z60 的完整算式;cmg.html 相位圖用的是「原始 z60」(非 MA20 平滑版)。
📈 8046 CMG 相位圖（X=NBC15_Z60 · Y=SBD_Z60）
個股:8046 南電 · 現值 NBC15_Z60=-0.66σ、SBD_Z60=-0.58σ · 象限 Q3 籌碼擴散 · 線=近10日軌跡、箭頭=速度 dR/dt。按「前一日/後一日」逐日回放(近 10 日,標記沿軌跡移動)。數值讀 FS broker_branch,換股只改 BLOCK_CMG_SYMBOL（本圖 8046）。
① NBC15_Z60 — X 軸 · 大資金方向
單日 NBC15 = ( ΣTop15淨買超分點的「淨買超」 − ΣTop15淨賣超分點的「淨賣超」 ) ÷ 當日總買量 × 100
NBC15_Z60 = rolling_z( 每日 NBC15 序列 , window=60 )
意義:有號的「淨集中度」—— 大資金淨買 vs 淨賣的力道差(占成交量 %)。Z60 > 0 = 中期偏吸籌;「持續站上 +1σ」比「單日 +3σ」更重要。
② SBD_Z60 — Y 軸 · 籌碼集中方向
單日 SBD = (淨賣超分點家數) − (淨買超分點家數)
SBD_Z60 = rolling_z( 每日 SBD 序列 , window=60 )
意義:正 = 賣家多買家少 → 籌碼集中於少數買方(吸籌);負 = 買家多賣家少 → 籌碼擴散(派發)。
共用 Z-score 公式
Z₆₀ = ( x(今日) − mean(x 近60日) ) ÷ std(x 近60日)
std = 樣本標準差(ddof=1,同 pandas rolling.std);資料不足 2 筆或 std=0 → 回 0;四捨五入 2 位。

四象限速記
四象限(X=NBC15_Z60, Y=SBD_Z60):Q1(+,+)主力吸籌 · Q2(−,+)承接觀察 · Q3(−,−)籌碼擴散 · Q4(+,−)短線推升。理想路徑 Q3/Q4 → Q1 主力吸籌。
③ Q1_power / Q3_power — 籌碼動力學 Level2 力量（60日）
前置　Rs(t) = EMA( R , 5 )　R=(NBC15_Z60, SBD_Z60)　計算用平滑·顯示用 raw
離心強度　|Rs| = √( x_Rs² + y_Rs² )
Q1_power₆₀ = Σ|Rs|（近60日 且 Rs∈Q1：x≥0 且 y≥0） ÷ 60
Q3_power₆₀ = Σ|Rs|（近60日 且 Rs∈Q3：x<0 且 y<0） ÷ 60
net_power = Q1_power − Q3_power　（>0 = 過去60日吸籌循環主導）
Qpower = Q1_power₆₀ − Q3_power₆₀　（吸籌淨力量;即上式 net_power）
意義:過去 60 天平滑軌跡在 Q1(吸籌)「停留時間 × 離原點強度」的綜合力量(每日平均);Q3 為派發力量。分母固定 60(整段視窗,非只除以落在該象限的天數)。程式碼:broker_branch.py:635 compute_cmg_kinetics · :668 Q1/Q3 累加 · :585 LOOKBACK=60 · :587 EMA_N=5。
別跟 q1_strength 混淆:q1_power 分母=60(整段)且用 EMA 平滑 Rs;q1_strength 分母=Q1 天數且用 raw R。近似 q1_power ≈ q1_strength × (q1_days ÷ 60)。

動力學其他欄位
─ compute_cmg_dynamics（raw + slope） ─
velocity (dx,dy) = (NBC15_Z60_slope, SBD_Z60_slope) σ/日
rotation_angle = atan2(dy,dx)°　（45°=朝 Q1 吸籌）
cmg_score = clip( (dx+dy)×150 + (x+y)×6 , −100..100 )　（≥30 吸籌轉強 / ≤−30 派發轉弱）
quadrant_path(_roman) = 走過象限（deadband 0.3 去抖, 反覆 1~2 象限標「震盪」）
─ compute_cmg_kinetics（EMA Rs） ─
V5 / V20 = Rs(t) − Rs(t−k)　（短=轉折 / 中=趨勢）
R·V = Rs · V5 內積　（>0 強化離原點 / <0 衰退回原點）
chip_momentum = net_power 側 × R·V → 吸籌強化/轉弱 · 派發強化/轉弱(賣壓耗盡)
ATS = net_power + Δnet_power(20日)　（吸籌品質是否正在變好）
rs_x / rs_y / rs_quadrant = EMA 平滑後的當前位置與象限
