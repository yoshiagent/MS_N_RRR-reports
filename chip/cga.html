💡 CGA — 資金重心引擎 · 選股評分邏輯詳解

CGA = 資金重心引擎 (Capital Gravity Agent)：量化「成交重心 VWAP20」相對「價格重心 MA20」的位置 / 方向 / 速度，輸出 Regime（6 類）＋ Signed Score（−100~+100）＋ Bias（5 tier）＋ Confidence。為 Box / Wyckoff / SHSL / RSR-RRR / Stock-Ops 提供「資金重心狀態」輔助訊號，本身不直接下買賣決策。

① 候選股池（先選哪些股來評分）：不是全市場海選，而是既有訊號的聯集 —— HotFlow（僅取 A 主流形成／B 加速，剔除 C 無感）∪ N-Pattern ∪ 自選配對 ∪ Stock-Ops。CGA 只對這池子逐檔算分、排序。

② 核心指標 VGO 族（皆 20 日窗，ATR 標準化 → 跨股可比）

MA20 ＝ 20 日收盤均價（價格重心）
VWAP20 ＝ Σ成交額₂₀ ÷ Σ成交量₂₀（成交重心＝市場平均持倉成本）
ATR20 ＝ 20 日均幅（波動標準化單位）
VGO_ATR ＝ (VWAP20 − MA20) ÷ ATR20　← 資金重心偏移（核心量）
VWAP20_SLOPE ＝ (VWAP20 − 5日前) ÷ ATR20（資金成本斜率）
MA20_SLOPE ＝ (MA20 − 5日前) ÷ ATR20（價格重心斜率）
VGO_VELOCITY ＝ VGO_ATR − 5日前 VGO_ATR（重心動能）　｜　VGO_ACCEL ＝ VELOCITY − 5日前 VELOCITY（加速度）

③ CGA Score 評分邏輯（signed −100 ~ +100）：五個分量各自歸一化並 clip 到 ±100，再加權求和後 clip：

分量來源指標歸一化 → ±100權重
C1 資金重心位置VGO_ATR× 10030%
C2 重心動能VGO_VELOCITY× 20025%
C3 資金成本斜率VWAP20_SLOPE× 33325%
C4 價站上成本CLOSE vs VWAP20上 +100 / 下 −10010%
C5 雙重心同向MA20_SLOPE × VWAP20_SLOPE同向 +100 / 反向 −5010%
Score ＝ 0.30·C1 ＋ 0.25·C2 ＋ 0.25·C3 ＋ 0.10·C4 ＋ 0.10·C5　（clip 至 ±100，四捨五入 1 位）

④ Bias 5 tier（由 Score 分級，門檻預設 UAA 可調）

Score 區間Bias

−70 ≤ Score < −30偏空
Score < −70強空

⑤ Regime 6 類（由上而下優先判定，命中即止）

Regime判定條件
📈 加速確認CLOSE > VWAP20 > MA20，且 VWAP、MA 兩斜率皆 > 0
🌱 吸籌偏多VGO_ATR > 0、動能 > 0、VWAP 斜率 > 0，且 MA 斜率 ≤ VWAP 斜率（資金領先價格）
🔄 回測成本VWAP20 > MA20、CLOSE 貼近 VWAP（相差 < 0.5·ATR20）、VWAP 斜率 ≥ 0
⚠️ 派發警示VGO_ATR 仍 > 0，但 CLOSE 跌破 VWAP、VWAP 斜率 ≤ 0、動能 < 0
📉 偏空下沉VGO_ATR < 0、動能 < 0、VWAP 斜率 < 0
⚪ 中性以上皆不符 / 指標資料不足

⑥ 三基礎狀態門檻（預設，UAA 可調）

資金重心位置：VGO_ATR > +0.30 → 資金在上 ⬆　｜　< −0.30 → 資金在下 ⬇
資金重心方向：VGO_VELOCITY > +0.15 → RISING　｜　< −0.15 → FALLING
資金成本趨勢：VWAP20_SLOPE > +0.10 → UP　｜　< −0.10 → DOWN

⑦ Confidence 信心：LOW ＝ K 棒 < 30 根／關鍵指標缺值／Box 無效；MEDIUM ＝ Regime 與 Score 方向背離（偏多 Regime 卻 Score < 0、偏空 Regime 卻 Score > 0、或 NEUTRAL 卻 |Score| > 50）；HIGH ＝ 兩者一致。

⑧ Box 軟依賴：若 box_structure 有箱體，額外用箱高標準化算 VGO_BOX 與 CLOSE／VWAP 在箱內位置（缺 Box 不影響評分，只少一組佐證）。

讀法：Score 給「多空強度與排序」，Regime 給「資金行為情境」；兩者一致（Confidence HIGH）時訊號最可靠，背離時回頭看 Regime 情境理解分歧原因。
