📐 方法學 / 三分類 / 門檻 / 陷阱（codex 顧問一輪收斂）
資料源：TWSE/TPEx OpenAPI t187ap11（董監事及持股 10% 以上大股東股權設質表），月更現況快照；每月 snapshot 累積成 pledge_snapshot_history.json（per-role 分桶 + top 質押人）。
偵測：月對月比較各檔「設質股數」下降 = 候選解質押。
三分類（核心）：
• 乾淨解質 clean_release：設質↓ 且董監持股沒明顯掉、無結構變動 → 財務壓力解除(偏多)。
• 疑似賣股還款 deleveraging_by_sale：設質↓ 但董監持股同期也↓（下降量 > 解質量的 50%）→ 可能只是賣股還款，甚至內部人降持股(中性偏空)。此為最大陷阱。
• 結構/身分變動 structural：董監總持股月變動 >±10% 且持股人數變動 ≥3（改選/增減資/名單變動）→ 不判讀。
雜訊閘門：解質規模 ≥500 張 且 ≥ 前月設質的 20%，否則濾除。
信心分級：乾淨解質 + (全數解除 / 脫離高風險區(比例 ≥50%→<30%) / 比例降 ≥15pp) → 高；降 ≥8pp → 中。
大股東 vs 董監經理人 分桶：依職稱欄（含「大股東」→ major_holder，其餘董事/監察人/經理人/法人代表 → insider）分開彙總；兩群經濟含義不同（大股東常為控股融資/經營權，董監偏內部人財務壓力）。
已知限制 / 待補：① MVP 無總股本，尺度用「設質股數 + 董監持股」相對衡量，「佔股本%」與跨股比較待正式版補 shares_outstanding；② 月快照只知「兩次之間有變」不知確切日期，不做精準事件日回測；③ 除權息/減增資造成的等比例股數變動，目前靠「結構閘門」粗略隔離，正式版應接股本/除權息資料精修；④ 姓名去重為 best-effort（同名不同人/法人代表）。
硬約束：歷史 <2 月不產生正式事件（codex）。累積後 + pool_history 回測證明「乾淨解質」真領先報酬，才考慮升級為選股/風控因子。目前僅每月留痕。
