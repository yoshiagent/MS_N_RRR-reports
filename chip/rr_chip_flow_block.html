🔀 RR 籌碼流向圖 — 配對 價×籌 相位圖

配對「價 × 籌」相位圖:X=價格相對伸展(配對價差 Z20)、Y=籌碼相對流向(雙方 Qpower 差)。只參考 chip/cmg_block.html 的 rolling_z 與 Qpower=net_power 定義建立。
當前讀數
配對A = 8046 ／ B = 3037
資料日2026-06-24
X · P_AB（Relative Price Stretch）+2.33 σ　= Z20( ln 8046/3037 )
Y · C_AB（Relative Chip Flow）+0.064　= Qpower(8046)=-0.259 − Qpower(3037)=-0.323
象限Q1 · 持續抱A（A過熱＋籌碼仍挺A）
X 軸 · Relative Price Stretch（價格相對伸展）
P_AB = Z20( ln( close_A / close_B ) )
衡量配對價差偏離 20 日均值幾個標準差。+2σ → A 相對過熱(A 漲太多或 B 太弱);−2σ → B 相對過熱。
Y 軸 · Relative Chip Flow（籌碼相對流向）
C_AB = Qpower_A − Qpower_B
Qpower = Q1_power60 − Q3_power60　（= net_power）
正 → 籌碼流向 A(A 吸籌力量較強);負 → 籌碼流向 B。
四象限
B 便宜 (PAB<0)A 過熱 (PAB>0)
籌碼流向A ↑ (CAB>0)Q2 觀察Q1 持續抱A
籌碼流向B ↓ (CAB<0)Q3 換B區Q4 警戒
Q1 持續抱A:A過熱+籌碼仍挺A → 趨勢強,續抱。　Q2 觀察:B便宜但籌碼流向A → 價籌背離,觀望。　Q3 換B區:B便宜+籌碼流向B → 雙確認,輪動到B。　Q4 警戒:A過熱但籌碼流向B → 背離警訊,留意A反轉。

解讀 / 換股路徑
理想換股路徑:Q1(抱A) → Q4(A過熱+籌碼轉弱·警戒) → Q3(換B);反向 Q3 → Q2 → Q1。|P_AB|≥2 = 過熱門檻(均值回歸機率升);|C_AB| 越大 = 雙方籌碼分歧越明顯。最佳換股訊號 = 價(P_AB)與籌(C_AB)同向確認(Q1 或 Q3);背離(Q2/Q4)宜觀望/警戒。
