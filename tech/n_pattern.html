N-Pattern — 策略與用法
📌 什麼是 N 型結構
「N」型走勢：底底高 → 突破 → 回測不破 → 再上攻。
代表股票剛脫離盤整、開始走多，具有「有底有頂」特性，
讓後續 RSR/RRR 配對的相對振盪更明顯、更可預測。
📊 5 維評分
總分 = 月線突破×25 + 週MA20×25 + 週底底高×20 + 回測MA20不破×15 + 日線量縮修正×15
滿分 100。
🟢 candidate / watch / reject
candidate：≥ 70 分（完整 N 型成立）
watch：50-69 分（部分維度未到位）
reject：< 50 分（不符合 N 型）
🎯 用法建議
candidate 的股票，下一步進 RR/RRR 找配對。
單一 N 型成立不是買進訊號——要等到能配上同產業另一檔（RR/RRR）才有資金輪動效益。
🔄 回測 MA20 不破
突破後回測 MA20，若 MA20 不破、且量縮修正，代表多頭結構未被破壞，是 N 型的關鍵確認點。
若回測破 MA20，N 型失敗，會降級為 watch 或 reject。
🔗 與其他 Agent 關係
上游：HotFlow A 級候選 → 進 N-Pattern 偵測
下游：N-Pattern candidate → RSR/RRR 配對
