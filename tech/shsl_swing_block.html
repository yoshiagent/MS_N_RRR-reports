定義
三時段(日/週/月)各取 SHSL Agent 的 wave_state + 4-flag(HH/HL/LH/LL) + 最後確認 SH/SL → 價位帶位置%
回檔箱(箱頂/箱底/世代/趨勢/位移)= box_structure agent 決定;報告頁只渲染(單一決策者)
Zigzag = 嚴格交替 SL→SH→SL… 近 6 點 + candidate(等確認);candidate 帶「3 條件確認進度」popover(跌破/回檔幅度/時間過濾)
判讀（波段位置）
wave_state:PULLBACK 拉回中(橘)· REBOUND 反彈中(紫)· BREAKOUT 突破(綠)· BREAKDOWN 跌破(紅)。4-flag 綠(HH/HL)= 多頭結構、紅(LH/LL)= 空頭結構。價位帶:SL(綠)→ 當前(黃)→ SH(紅),位置% = 在區間內位置(越高越接近壓力)。三時段同步(皆 REBOUND/PULLBACK)= 共振訊號;週反彈+日拉回 = 可能買點。⛔ 週線下降箱(LH+LL)= 紀律規範不得操作、持有應出清。
