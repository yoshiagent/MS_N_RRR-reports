🧭 價量(金額)相位 — 成交金額增減幅 × 漲跌幅 · 8046 南電

🧱 B25 · 📊 圖

📊 四象限 Rolling QPower(期間主導力量·0~100)
點圖或按鈕選日
QPower = 50%位置 + 30%持續 + 20%速度 的綜合力量,再取 EMA(20) 平滑;哪條線在上=近期哪種價量型態主導。Instant(當下)與 Rolling(期間)引擎皆有輸出,圖用 Rolling、結論卡兩者並列。
⚖ Q1−Q3 淨力量(＋= 價量齊揚強勢 · − = 量縮價跌弱勢)
點圖或按鈕選日
參考線 ±20(DOMINANT 主導)/±5(LEANING 略優)/0;由正翻負或跌破 +5,搭配相位圖落 Q2/Q4,即「多頭轉弱 / 出貨」的量化對照。
🗺 四象限意義(經典價量關係·CIO 定義)
象限價量狀態傾向解讀
Q1 (+量,+價)量增價漲價量齊揚資金進場、健康上攻 → 續強
Q2 (−量,+價)量縮價漲量縮價漲惜售/背離,追高動能減弱 → 上漲末端警訊
Q3 (−量,−價)量縮價跌量縮價跌賣壓衰竭、縮量整理 → 可能打底
Q4 (+量,−價)量增價跌量增價跌出貨、恐慌賣壓 → 危險
CENTER|z| 兩軸皆 < 0.10(dead zone)中軸區不下象限判斷(防零軸抖動)
TRANSITION單軸 < 0.10轉換帶象限歸屬不穩定,等突破灰帶
X 正向 = 成交金額放大(相對近 20 日環比·成交強度提升);Y 正向 = 收盤上漲(相對近 20 日環比·價格反應)。健康多頭常沿 Q3→Q1(縮量打底 → 放量上攻)、頭部常見 Q1→Q2→Q4(價量齊揚 → 背離 → 出貨)。對角 Q1(強勢健康)↔ Q3(弱勢);Q2(背離)↔ Q4(出貨)。

定義與公式鏈(4Q 引擎 V1)
amount = close × volume　X = 成交金額增減幅 = (amount_t / amount_{t−1} − 1) × 100%　Y = 漲跌幅 = (close_t / close_{t−1} − 1) × 100%
x = rolling_z(X, 20)　y = rolling_z(Y, 20)　(各自對近 20 個環比值標準化·ddof=1·同 B1a/B24 口徑)
Rs(t) = EMA(z, 5)　V(t) = Rs(t) − Rs(t−1)　A(t) = V(t) − V(t−1)
象限:|Rs| 兩軸 < 0.10 → CENTER;單軸 → TRANSITION;否則依 Rs 正負落 Q1~Q4
score_Qk = clip( max(dot(向量, u_Qk), 0) ÷ cap, 0, 1 ) × 100　u_Qk = 象限中心單位向量(±0.7071)
Instant QPower_Qk = 0.50×位置分 + 0.30×持續分 + 0.20×速度分　Rolling QPower_Qk = EMA( instant , 20 )　Q1−Q3 淨力量 = Q1_rolling − Q3_rolling
