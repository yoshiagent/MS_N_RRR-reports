HotFlow — 策略與用法
🔥 什麼是 HotFlow
偵測「全市場主流形成」的 Agent。對 ~1,935 檔股票打分，把當前資金流向最熱的族群與個股篩出來。
不做買賣決策，僅作為下游 N-Pattern / RR / RRR 的候選池。
📊 HotFlowScore 算法
11 個衍生欄位 → percentile 排名 → winsorize 抑制離群 → 加權平均：
月漲幅×0.25 + 週漲幅×0.20 + 5日量比×0.15 + 月底站MA×0.15 + 法人連買×0.15 + 上影壓力×0.10
回傳 0~100 分。
🟢 A 級條件
A 主流形成中：HotFlowScore ≥ 70
且符合：月線突破近 12 月高 / 週 K 站上 MA20 / 量能擴張
通常為下游 RR/RRR 配對的優先候選。
🟡 B 級觀察 / 🔴 C 級排除
B 觀察中：60-70 分，部分條件未齊全
C 過熱或排除：< 60 分，已飆漲完畢或趨勢轉空

收合區塊可分別展開檢視。
🎯 用法建議
不要直接買 A 級。HotFlow 只告訴你「主流在哪」，具體進場時機要看：
① N-Pattern 是否成立
② RR/RRR 是否有可配對輪動的對手
③ 市場結構是否允許（regime ≥ 中性）
🔄 資料新鮮度
每日 18:00 排程器自動更新（TWSE/TPEX 17:30 公布後）。
若主頁顯示 as_of 與當天不一致，代表 fetch 階段網路逾時，可手動跑 python -m src.agents.hotflow。
