💡 操作建議 (Operations Guide)
本頁三池只是「初篩」，進場前還需通過：(1) 市場結構分（多頭/警戒/防禦）、(2) 操作 Agent 5 階段判斷、(3) MS-Agent v2 風險閘。單純按本池追股 = 忽略大環境風險。
🏆 用法 1：長期核心持倉
從 🏆 Both 雙料池挑 3-5 檔。
✓ 既強又有續航力，可承受 5-15% 回檔不出場
✓ 適合大部位 (30-50% 資金)
✓ 建議：每檔等權配置，定期 (週/月) 檢視族群分數變化
本日代表：永豐金 ／ 元大金 ／ 富邦金
📈 用法 2：波段追熱
從 📈 Core 主流核心池挑（避開 Both 池重複）。
✓ 順勢追當前最強族群
✓ 風險：Future 分低 = 可能短期內轉弱 → 設緊停損 (-5%)
✓ 適合 10-20% 資金，勿超出 2 週持有
本日代表：霖宏 ／ 菱生 ／ 百容
🌱 用法 3：分批布局新主流
從 🌱 Candidate 候選池分批進場（避開 Both 池重複）。
✓ 早期上車，但需等 Current Score 升上來確認
✓ 建議：分 3 批進，每批間隔 5-10 日
✓ 適合 5-15% 資金，容忍 1-2 個月磨底
⚠ 若 30 日後 Current Score 仍未升 = 結構未成型 → 出場
本日代表：日月光投控 ／ 智原 ／ 寶雅
⚠ 用法 4：避免追高 (排除清單)
不要追 Core 池中 Future Score < 20 的個股。
✓ 雖然目前強，但未來分極低 = 高機率即將轉弱
✓ 若已持有 → 開始分批減碼
✓ 若未持有 → 直接跳過，找 Both 或 Candidate
本日警示：（本日 Core 池無此情形）
⚠ 重要風險提醒
  • 市場結構優先：若主頁「市場結構」顯示「防禦」或「出場」，本頁三池全部失效 → 降低持股比重至 0-25%
  • regime change 會打亂歷史 pattern：多頭→空頭轉折時，過去的領跑族群可能突然失效
  • 樣本量限制：1 年歷史 ≈ 75 次主導族群轉移，預測準確度約 50-65%
  • 個股仍需個別驗證：本頁挑出的股，進場前仍需用操作 Agent看 5 階段判斷與 A/B/C 風險
  • 勿單押一池：建議 Both + Core + Candidate 三池資金比約 5:3:2，避免風格集中
🔗 與系統其他工具的搭配流程
1. 看主頁市場結構 → 確認 regime 是否允許進場（多頭/警戒以上）
2. 看 MS-Agent v2 → 確認市場溫度 + RR 環境 + 風險 OK
3. 本頁主流股池 → 選 Both / Core / Candidate 候選
4. 用 操作 Agent (Stock Ops) → 確認該股的 5 階段建議 + A/B/C 風險
5. 用共振配對或自選配對 → 找出該股的對沖配對（多/空 RSR）
6. 最終下單前，用族群輪動確認該族群仍在「Improving/Leading」軌跡上
