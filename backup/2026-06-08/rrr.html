RRR 共振輪動 — 策略與用法
🔄 什麼是 RRR
Relative Resonance Rotation（相對共振輪動）— 嚴格版。
RR + cycle 規律性 + 雙邊極端折返 + 風控閘三層篩選，
追求「高勝率、低頻換股」的中長波段配對。
📊 三維評分
cycle　Z60 在視窗內的零軸交叉次數（規律性）
phase　當前 Z60 位置（極端 / 中段 / 對稱）
mean_reversion　雙邊是否曾觸 ±2 極端

合計 ≥ 70 才為 tradable。
✅ C1 換股確認
Z60 突破 ±2 後不立刻換股，
等折返進入緩衝區 ±0.3（或 dynamic Z）才確認。

目的：避免在 Z=+3 還會再衝高時誤判進場。
🛡 風控閘 (risk_control)
所有 switch 訊號最終須通過 risk_control.gate()：
① 市場狀態 ≥ defensive
② 兩檔籌碼非 reject
③ confidence ≥ 50
④ 部位上限依市場 state（25/50/75/100%）
📑 報告 4 區塊
最終行動建議　風控放行的 switch
換股訊號　所有 switch 配對
續抱訊號　hold_A / hold_B（次要）
風控阻擋　被 veto 的訊號（含原因）
🎯 用法建議
RRR 是系統的「主訊號」。
看「最終行動建議」第一張卡：
① 部位 size（建議入場資金比例）
② 賣 → 買 的兩檔
③ Z60 + confidence
主頁也只顯示 RRR 放行的訊號，不顯示 RR。
