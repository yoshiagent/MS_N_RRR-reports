定義
主力買賣超 = FinMind Sponsor「真分點」券商每日買賣超 top（🌐 = 外資券商）
外資 vs 本土 net = 外資/本土券商淨買賣超合計 → fd_signal 方向
🔒 點石神 = 連續居買超 top 的券商；⚡ 隔日沖 = 買後隔日賣出；👣 累積足跡 = 某分點連續至最新日每日淨買超(≥3 日, Wyckoff 吸籌微觀證據)
SBD = 賣家數−買家數(正=賣方分散買方集中=主力吸籌)；換手品質 = 漲多後主力出貨時, 接手者是新主力(連買) 還是散戶
判讀（主力辨識）
外資/點石神/累積足跡 同向買超 + SBD 正 + 籌碼集中 = 主力吸籌(偏多);隔日沖多、換手品質差(散戶接手)、SBD 負 = 籌碼鬆動(偏空)。分點是『誰在買賣』的微觀證據, 須與價量/法人/融資合看, 非單獨進出訊號。
