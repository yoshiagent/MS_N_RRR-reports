🔬 NBC15_Z20 · SBD_Z20 · Qpower20 公式說明（四象限 · z20/z60 對照 · 程式位置）　▸ 點開
① NBC15_Z20 — X 軸 · 大資金方向 (短線)
單日 NBC15 = ( ΣTop15淨買超分點的「淨買超」 − ΣTop15淨賣超分點的「淨賣超」 ) ÷ 當日總買量 × 100
NBC15_Z20 = rolling_z( 每日 NBC15 序列 , window=20 )
意義:有號的「淨集中度」—— 大資金淨買 vs 淨賣的力道差(占成交量 %)。z20 窗短、轉折快;「近 20 日站上 +1σ」即偏短線吸籌(vs z60 看中期是否成趨勢)。
② SBD_Z20 — Y 軸 · 籌碼集中方向 (短線)
單日 SBD = (淨賣超分點家數) − (淨買超分點家數)
SBD_Z20 = rolling_z( 每日 SBD 序列 , window=20 )
意義:正 = 賣家多買家少 → 籌碼集中於少數買方(吸籌);負 = 買家多賣家少 → 籌碼擴散(派發)。z20 = 近 20 日的集中/擴散短線相位。
共用 Z-score 公式 (窗=20)
Z₂₀ = ( x(今日) − mean(x 近20日) ) ÷ std(x 近20日)
std = 樣本標準差(ddof=1,同 pandas rolling.std);資料不足 2 筆或 std=0 → 回 0;四捨五入 2 位。窗=20(與 z60 唯一差別)。
四象限速記
四象限(X=NBC15_Z20, Y=SBD_Z20):Q1(+,+)主力吸籌 · Q2(−,+)承接觀察 · Q3(−,−)籌碼擴散 · Q4(+,−)短線推升。理想路徑 Q3/Q4 → Q1 主力吸籌(與 z60 象限語意相同,窗較短)。
③ Qpower20 / CAB20 — 籌碼動力學 (20日·主力2D·z20 口徑)
前置　Rs(t) = EMA( R , 5 )　R=(NBC15_Z20, SBD_Z20)　計算用平滑·顯示用 raw
離心強度　|Rs| = √( x_Rs² + y_Rs² )
Q1_power₂₀ = Σ|Rs|（近20日 且 Rs∈Q1：x≥0 且 y≥0） ÷ 20
Q3_power₂₀ = Σ|Rs|（近20日 且 Rs∈Q3：x<0 且 y<0） ÷ 20
Qpower20 = Q1_power₂₀ − Q3_power₂₀　（短線吸籌淨力量;即 net_power·窗20）
CAB20 = Qpower20(A) − Qpower20(B)　（配對淨力差;≥+0.05 持A / ≤−0.05 持B）
意義:過去 20 天平滑軌跡在 Q1(吸籌)「停留時間 × 離原點強度」的綜合力量(每日平均);Q3 為派發力量。分母固定 20(整段視窗,非只除以落在該象限的天數)。Qpower20 就是精簡輪動卡「主力(2D·z20)」的每股力量,兩股相減 = CAB20;B22 價差K線第三格畫的「線」即 CAB20。
z20 vs z60 差在窗:z20 反應快、雜訊多(短線換股時機用);z60 穩、慢(中期吸籌趨勢用·B1)。NBC 易假突破 → 系統要求「NBC 同向 ＋(OCS 或 CAB 至少一個附和)」才算數。
