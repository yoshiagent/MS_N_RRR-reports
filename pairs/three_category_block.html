1 · 相對價格（價差反轉觀點·＝配對均值回歸前提）：Z20 = zscore(ln A/B)。Z20 ≥ +1.0σ → A 價差過熱 → 賭收斂、持便宜側 B；Z20 ≤ −1.0σ → 持 A；|Z20| < 1.0 → 中性。（過熱側該減、持便宜側；離帶折返＝時機）
2 · 主力籌碼（跟大資金方向·NBC 為主 SBD 為品質濾網）：以 2D 方向票的 NBC（大資金方向差 = rs_x_B − rs_x_A）定持有——NBC ≥ +0.1 → 持 B；≤ −0.1 → 持 A；|NBC| < 0.1 → 中性。SBD（集中度）定品質：散戶擴散 或 大資金反向 → 信心降級；假突破 → veto。⚠ 注意 net(CAB 純量) 與 NBC 大資金方向可能背離（散戶驅動）→ 以大資金方向為準。
3 · 相對價籌（PAB×CAB 象限直接給）：Q1(A貴+籌挺A) 持 A｜Q3(B便宜+籌挺B) 持 B｜Q4(A貴+籌流B·A頂背離) 換 B 醞釀(未確認)｜Q2(B便宜+籌挺A) 持 A 觀察(勿追 B)。
三類共識：三類全同 → 一致（可信度高·綠）；分歧 → 等確認·別急換（橙）；多中性 → 觀望。三個獨立源（價 / 籌 / 價籌）一致才有信心，防接刀。門檻（±1.0σ／±0.1）為 v1 暫定，待回測校準。
