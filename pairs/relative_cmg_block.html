🧭 Relative CMG — ΔNBC15_Z20 × ΔSBD_Z20 · 南電(8046) vs 欣興(3037)

🧱 B24 · 📊 圖 · 引用 B1a

📊 四象限 Rolling QPower(期間主導力量·0~100)
點圖或按鈕選日
QPower = 50%位置 + 30%持續 + 20%速度 的綜合力量,再取 EMA(20) 平滑;哪條線在上=近期哪個象限主導。Instant(當下)與 Rolling(期間)引擎皆有輸出,圖用 Rolling、結論卡兩者並列。
⚖ Q1−Q3 淨力量(＋= 偏 南電 力量 · − = 偏 欣興 力量)
點圖或按鈕選日
參考線 ±20(DOMINANT 主導)/±5(LEANING 略優)/0;由正翻負或跌破 +5,搭配相位圖落 Q4,即「A 優勢消失 → 輪動前夕」的量化對照。
🗺 四象限意義(CIO 定義·相對差平面)
象限相對籌碼狀態傾向解讀
Q1 (+,+)南電 比 欣興 更集中、更吸籌偏向 南電A 雙優 → 續偏 A 方
Q2 (−,+)還沒集中、已開始吸籌資金轉向 南電資金開始轉向 A → 早期換股訊號
Q3 (−,−)南電 比 欣興 分散、派發偏向 欣興B 雙優 → 續偏 B 方
Q4 (+,−)還很集中、但開始派發南電 優勢消失A 優勢消失 → 常見於 南電→欣興 輪動前夕
CENTER|Δ| 兩軸皆 < 0.10(dead zone)中軸區不下象限判斷(防零軸抖動)
TRANSITION單軸 < 0.10轉換帶象限歸屬不穩定,等突破灰帶
X 正向 = 南電 比 欣興 更「集中」(大資金淨買力道差);Y 正向 = 南電 比 欣興 更「吸籌」(籌碼集中家數差)。理想輪動路徑:Q3 → Q2(早期訊號)→ Q1(A 雙優)→ Q4(A 優勢消失)→ Q3。
