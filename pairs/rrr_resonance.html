💡 方法說明 — 雙因子去因子化 + 共振配對偵測

🎯 目標：從主流股池找出「週期接近 + 相位約 180°」的配對股 — 中週期反相波動，可用於對沖 / 輪動換股 / 共振交易。

⚙ 方法（兩層流水線 + 強化）：

Layer 0 [v3 雙因子] · 市場去因子化（TWII + OTCI） — 對每股做多元 OLS：

r_stock = α + β_TWII · r_TWII + β_OTCI · r_OTCI + ε

殘差 ε 才是「扣除大盤 + 中小型/科技因子後的純個股波動」。追蹤 獨立性 = 1 − R²（個股獨立變異占比）— 需 ≥ 20% 才參與配對（沒輪動空間的股票排除）。

Layer 1 · 對每股「殘差 returns」用 autocorrelation 找主週期 T；配對要求 |T_A − T_B| / max(T) ≤ 20% 且 殘差相關 ρ ≤ 0.2

Layer 2 · Butter 帶通（中心 1/T_avg ± 30%）+ Hilbert 取瞬時相位

PLV = | mean(exp(i·(φ_A − φ_B))) | ≥ 0.55
|circmean(Δφ) − π| ≤ 36°

Layer 3 [v2 新增] · 滾動穩定性 — 把 lookback 切 3 段，要至少 2/3 段 PLV 達標才算「穩定」

🔄 與既有 RRR (Z60) 的差異：
• 本 Agent = 系統性配對選股（找哪些股有共振關係）
• 既有 RRR (Z60) = 觸發時機（在已知配對上找入場點）
兩者互補。

篩選門檻： 週期容忍 ≤ 20% ｜ 殘差相關 ≤ 0.2 ｜ 獨立性 ≥ 20% ｜ PLV ≥ 0.55 ｜ |Δφ−180°| ≤ 36° ｜ 週期範圍 5-40 天 ｜ 滾動 3 窗 達標 ≥ 67% ｜ 市場因子（TWII+OTCI）= ✅ 開啟
