💡 怎麼讀這張表？

1. 差異 (B−A)：啟用 RCGA filter 多賺多少 — 正值越大代表 filter 越有用

2. 換股 ／ 攔截：純 Z60 換股次數 ／ 其中被 RCGA 攔下的 (A_DOMINANT 或 B_DOMINANT)

3. 滾動穩定性：12 個滾動窗中 RCGA 勝 / 平 / 負 的窗數 — 重點看「負」是不是 0，0 表示從不傷害

4. 建議分級：

✅ 強烈建議 = 0 負 + 有勝🟢 建議 = 勝率 60%+🟡 可選 = 勝率 50%+⚪ 中性🔴 不建議

set PAIR_RCGA_FILTER=1　　(Windows)
$env:PAIR_RCGA_FILTER=1　 (PowerShell)
