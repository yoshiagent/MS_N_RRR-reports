🗺 這個 block（B13 · RVGO 相對成交重心 — 定義 · 判讀 · 門檻 · 資料 · 程式 — 點開）
定義（公式鏈：OHLCV → vgo_atr → RVGO）
TP = (high+low+close)/3　|　MA20 = SMA(close,20)
VWAP20 = Σ(amount,20) / Σ(volume,20)　（amount 缺 → TP×volume）
ATR20 = ATR(20)
vgo_atr = (VWAP20 − MA20) / ATR20　← 單股「成交重心」(資金成本相對均價的偏離, ATR 單位)
vwap20_slope = (VWAP20 − VWAP20[−5]) / ATR20　← 單股資金成本上升速度
RVGO 系列（配對相對化）
RVGO　　　　= vgo_atr(南) − vgo_atr(欣)　（成交重心差, ATR 單位）
RVGO velocity = RVGO(t) − RVGO(t−5)　（相對資金重心的移動方向）
RVWAP slope　= vwap20_slope(南) − vwap20_slope(欣)　（兩股資金成本上升速度差）
RVGOZ20　　= (RVGO − MA20(RVGO)) / STD20(RVGO)　（RVGO 偏離常態幾 σ）
判讀（資金重心偏哪邊）
RVGO > 0 → 成交重心偏 南（南電）;< 0 → 偏 欣（欣興）。RVGO velocity > 0 → 資金「正在」流向 南;< 0 → 流向 欣。RVWAP slope > 0 → 南 的 VWAP20 上升較快(短期動能強)。三者同向 = 高信心;分歧（如 RVGO 仍正但 velocity/slope 轉負）= 動能正在衰減的早期警訊。
門檻（RCGA Ver.1 預設）
RVGO：> +0.3 偏 A（紅） / < −0.3 偏 B（藍）
velocity：> +0.15 / < −0.15　·　RVWAP slope：> +0.1 / < −0.1
|RVGOZ20| ≥ 1.5 警戒 / ≥ 2 極端（與 Z60 同 Z-score 邏輯統一）
為什麼跟 Z60 互補
Z60 看的是『價格』偏離（ln 價差的統計極端），弱點是漲多時誤判極端;RVGO 系列看的是『主力資金的偏好結構』（VWAP vs 均價）。換股本質是賭收斂,但收斂前提是動能衰減 — Z60 看不到動能,RVGO 正好補上。例:Z60>+2 但 RVGO velocity 仍 >0 → A 動能未衰,宜延緩換股。
