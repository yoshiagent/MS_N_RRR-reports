四、資料流（每日執行細節）

17:30  TWSE/TPEX 三大法人公布
17:40  TAIFEX 期貨/選擇權公布
18:00 ────────────────────────────────────────
       Windows Scheduler 觸發 MS_N_RRR_Daily

       Step 1: Data Agent fetch
                stock_data.fetch_all(symbols=DEFAULT_UNIVERSE,
                  sources=["price", "index", "macro", "3insti", "futures",
                           "market_insti", "options", "etf"])
                → 寫入 stockHisData/processed/

       Step 2: Market Structure
                evaluate_market_structure()
                  ├─ TWII 日/週/月、OTCI 週
                  ├─ FX、VIX
                  ├─ 外資/投信/自營 台指期 OI（各自評分）
                  └─ structure_zscore.compute_structure_zscore()
                       └─ 60d Z: foreign_spot, foreign_fut, usdtwd, sox,
                                 trust, hidiv_vol, mkt_dd_narrow
                       → foreign_pressure_score, etf_support_score,
                         market_structure_score, regime, dynamic_z_threshold
                → market.json

       Step 3: HotFlow
                全市場 ~2,700 → A/B/C 分類
                → hotflow.json + hotflow.html

       Step 4: N-Pattern
                從 HotFlow A+B 取候選 → N 型結構評分
                → n_pattern.json + n_pattern.html

       Step 5a: RR Pair
                簡版 RS_MA20 穿越
                → rr.json + rr.html

       Step 5b: RRR Pair + RS/Z-score
                rs_zscore.compute_signal()
                  └─ 讀 market.json 的 dynamic_z_threshold
                     （regime=strong_bull+wma20→2.5, neutral→2.0, mild_bear→1.5）
                嚴格版 = cycle + phase + mean_reversion + Z60 折返
                → rrr.json + rrr.html

       Step 6+7+8: Chip → Risk → Report
                每個 RRR 訊號跑 risk_control.gate()
                合成 report.html / index.html
                → 含 1) market structure  2) Agent 卡片 × 5
                       3) 最終行動建議  4) 詳細頁連結
                       5) 7 日 backup 連結  6) 系統文件清單

       Step 9: Custom Pairs
                evaluate_custom_pairs()
                  ├─ 讀 配對表.xlsx（自選 sheet）
                  ├─ 依序兩兩成對
                  ├─ 每組跑 rs_zscore.compute_signal()
                  └─ 每組呼叫 pair_chart.build_for_pair()
                     → pair_{a}_{b}.html（5 組配對的完整 Plotly 圖）
                → custom_pairs.json + custom_pairs.html

       Step 10: Backup snapshot + 7 日清理
                src/shared/backup.py

       Step 11: Deploy
                scripts/deploy_reports.py
                  └─ git -C data/output push origin main
                  → https://yoshiagent.github.io/MS_N_RRR-reports/

       Step 12: Notify
                src/notify/__init__.send_daily()
                  ├─ Telegram HTML
                  └─ Email
