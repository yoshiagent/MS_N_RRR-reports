Chip Agent（籌碼濾網）

任務

作為「濾網」與「警示」用途，不作為主要買賣依據。 協助過濾掉籌碼明顯轉壞、或有重大借券 / 大戶出貨跡象的個股。

觀察項目

  • 外資買賣超（單日 / 5 日 / 20 日累計）
  • 投信買賣超（單日 / 5 日 / 20 日累計）
  • 自營商買賣超（單日 / 5 日 / 20 日累計）
  • 主力買賣超（前 15 大券商分點淨買賣）
  • 借券餘額變化（借券大幅增加為警示）
  • ETF 成分股調整對資金的影響

輸入資料

  • 三大法人買賣超（日資料）
  • 券商分點資料（如可取得）
  • 借券餘額日資料
  • ETF 成分股清單與權重（如台灣 50、台灣中型 100、各主題 ETF）

輸出格式

{
  "symbol": "",
  "chip_score": 0,
  "foreign_score": 0,
  "investment_trust_score": 0,
  "dealer_score": 0,
  "main_force_score": 0,
  "warning": [],
  "status": "support / neutral / warning / reject"
}

狀態定義

  • support：籌碼明顯支撐（多項法人同步買超 + 主力買盤）
  • neutral：無明顯訊號
  • warning：出現警示但尚不致命（如借券大增）
  • reject：籌碼明顯轉壞（法人連續大賣 + 主力分點同步出貨）

否決條件

  • 連續多日外資大幅賣超且伴隨價跌
  • 借券餘額短期內暴增，且股價已轉弱
  • 主力分點明顯轉為淨賣方且集中度提高

與其他 Agent 的關係

  • 是濾網，不主動產生買賣訊號
  • 對 rs_zscore_agent 的 switch 訊號做 veto
  • 對 risk_agent 提供倉位降級依據

未來擴充方向

  • 加入大戶持股比例變化
  • 加入融資融券餘額（過熱與斷頭風險）
  • 引入 ETF 被動買賣對個股流動性影響的估算
  • 主力分點分群（短線 / 中長線 / 外資分點）細分
