N-Shape Agent

任務

從股票池中找出具有「N 型波段結構」的強勢股，作為 RRR 配對的候選池。 重點是抓「剛脫離盤整、開始走多」的階段，而不是追已飆漲完畢的股票。

N 型結構定義

  1. 月線剛脫離長期盤整區（突破或站穩盤整箱頂）
  2. 週線形成底底高（higher low）
  3. 週 MA20 向上
  4. 股價回測週 MA20 不破且止穩
  5. 日線呈現量縮修正，不在追高位置

輸入資料

  • 個股日線、週線、月線 OHLCV
  • 個股 MA5 / MA20 / MA60 / MA120
  • 股票池（來自 config/stock_pool.yaml）
  • 市場狀態（來自 market_structure_agent，決定是否啟動掃描）

評分維度（總分 0-100）

  • 月線脫離盤整強度
  • 週線底底高完整度
  • 週 MA20 斜率
  • 回測週 MA20 是否成功
  • 日線修正量能（縮量為佳）

輸出格式

{
  "symbol": "",
  "n_shape_score": 0,
  "timeframe": "monthly / weekly / daily",
  "reason": [],
  "status": "candidate / watch / reject"
}

否決條件

  • 月線仍在下降趨勢
  • 週線跌破前波低點（破壞底底高）
  • 週 MA20 向下
  • 日線在追高位置（距 MA20 過遠）
  • 量價背離（價漲量縮且型態走完）

與其他 Agent 的關係

  • 上游：market_structure_agent（非 bullish/warning 時可選擇不啟動）
  • 下游：rrr_pair_agent（只在 N-shape 候選池中配對）
