整合 4：MS-Agent v2（市場結構監控中心）

來源

  • 使用者規格 — 「不預測漲跌，持續監控資金如何流動」
  • 6 模組架構（M1-M6）+ Market Temperature + RR Environment + Market Risk

提供的能力

  1. 雙軌→三軌 sector_map — 官方 28 類 + 自建主題 + XQ 545 細產業（含真實市值）
  2. 市場 Breadth (M2) — 4 指標合成 0-100 分（healthy/mixed/weak/panic）
  3. 資金擴散 (M3) — HHI（用 XQ 細產業）+ 新進榜 + 中型佔比（用 XQ 真實市值）
  4. HotFlow lifecycle (M4) — 形成/擴散/過熱/退潮/靜止 5 狀態機 + 跨日 state + 歷史回放
  5. 市場溫度 — 8 因子加權合成（5 級分類）
  6. RR 環境分 — 6 條件加總（3 級分類，影響 dynamic_rr）
  7. 大盤層級風險 (M6) — 7 訊號偵測 + 部位上限自動調整

設計重點

  1. 資料層整合：XQ industry_sub.csv（big5）由使用者匯出，每月手動更新；其他資料由 OpenAPI 自動抓
  2. HHI 主鍵用 XQ 細產業而非官方 28 類：因半導體+電子零組件天然佔台股大半，用官方算永遠 over_concentrated 失去敏感度
  3. 市值用 XQ 真實值：取代之前的「排名 proxy」（之前用 rank≥21 假定為中小型，錯誤）
  4. lifecycle state 持久化：跨日追蹤、自動清理 30 日 stale 檔
  5. 獨立頁不取代主頁：第 7 個子頁 ms_dashboard.html，主頁加入口卡（保守整合，可隨時改取代）
  6. as_of 邏輯：選「最近一個全市場齊全（≥1000 檔）」交易日，避免取到只有 universe 子集的不完整日

Phase 1：演算法移植（commit c3f7979）

目標：把 60 日 Z-score 模型搬進 Ver.A，但資料源暫讀姊妹專案 raw CSV。

搬入位置：src/shared/structure_zscore.py

演算法：

foreign_pressure_score = -0.30*Z_foreign_spot + -0.30*Z_foreign_fut
                       + +0.20*Z_usdtwd + -0.20*Z_sox

etf_support_score      = +0.35*Z_trust + +0.25*Z_etf_holder
                       + +0.20*Z_hidiv_vol + +0.20*Z_mkt_dd_narrow

market_structure_score = etf_support_score − foreign_pressure_score

regime: > +1.0 strong_bull / +0.3~+1.0 mild_bull / -0.3~+0.3 neutral
         / -1.0~-0.3 mild_bear / < -1.0 strong_bear

dynamic_z_threshold:
  strong_bull + 站上週MA20 → 2.5
  neutral                  → 2.0
  < neutral 門檻            → 1.5

接入： - src/agents/market_structure.py.evaluate_market_structure() 末段呼叫 compute_structure_zscore() - 結果合併進 market.json 的 regime / foreign_pressure_score / etf_support_score / market_structure_score / dynamic_z_threshold 等欄位 - src/agents/rs_zscore.py.compute_signal() 讀 market.json 的 dynamic_z_threshold 覆蓋靜態 ±2.0 - src/agents/index_builder.py._render_extended_market() 顯示在進階指標 banner

START = "2025-01-01"

價差 = 南電收盤 − 欣興收盤
MA60 = 價差 60 日均值
SD60 = 價差 60 日標準差

# C1 換股確認機制

if Z60 < -1.5 and 當日 Z60 > 前日 Z60: 換成南電（A）
