📜 XS 選股腳本原始碼 — 月MACD金叉起漲（XQ 可直接複製 — 點開）
// =====================================================================
// 月MACD金叉起漲 選股 (3P Ⓑ 嚴格版; 原名 三位一體共振選股)
//   月：MACD 黃金交叉 + 柱狀圖(OSC)增長
//   週：底底高（最近 8 週低點 > 前 8 週低點）
//   日：MA20 向上 + 股價站上 MA20 + 近 5 日回測 MA20
// 適用：選股腳本 (Scanner)
// =====================================================================

// 月 MACD(12,26,9) 需 ~80 個月歷史 ≈ 80*22 = 1760 日，設 2000 安全
SetTotalBar(2000);

// === 變數宣告 ===
// MACD 三個 by-reference 輸出
var: difValue(0), macdValue(0), oscValue(0);
// 各週期條件旗標
var: condM(false), condW(false), condD(false);
// 週線波段低點
var: wL1(0), wL2(0);
// 日線 MA20 與 5 日前 MA20（用來判斷向上）
var: ma20(0), ma20Prev(0);

// =====================================================================
// 條件 1：月線 MACD 黃金交叉 + 柱狀圖增長
// =====================================================================
// 以月線收盤序列計算 MACD，dif/macd/osc 自動更新為「最新月」的值
MACD(GetField("收盤價", "M"), 12, 26, 9, difValue, macdValue, oscValue);

// 黃金交叉：DIF 向上穿越 MACD 線
// 柱狀圖增長：OSC = DIF - MACD 比上月放大
condM = (difValue Crosses Above macdValue) and (oscValue > oscValue[1]);

// =====================================================================
// 條件 2：週線底底高
// =====================================================================
// WL1 = 最近 8 週的最低 Low
// WL2 = 再往前 8 週（9~16 週前）的最低 Low
wL1 = Lowest(GetField("最低價", "W"), 8);
wL2 = Lowest(GetField("最低價", "W")[8], 8);
condW = wL1 > wL2;

// =====================================================================
// 條件 3：日線 MA20 向上 + 站上 + 回測
// =====================================================================
ma20     = Average(Close, 20);
ma20Prev = Average(Close, 20)[5];   // 5 日前的 MA20

// (a) MA20 向上：今日 MA20 > 5 日前的 MA20
// (b) 股價站上 MA20：今日收盤 > MA20
// (c) 回測 MA20：近 5 日內最低點曾觸碰過 MA20 附近（容差 0.5%）
condD = (ma20 > ma20Prev)
    and (Close > ma20)
    and (Lowest(Low, 5) <= ma20 * 1.005);

// =====================================================================
// 共振：三條件全成立才觸發
// =====================================================================
if condM and condW and condD then begin
    ret = 1;
    // order:=1 → 此欄位作為預設排序鍵（由大到小）
    OutputField1(GetField("成交金額(億)", "D"), "成交金額(億)", order:=1);
    OutputField2(oscValue,                     "月MACD柱");
    OutputField3(wL1,                          "週WL1");
    OutputField4(wL2,                          "週WL2");
    OutputField5(ma20,                         "日MA20");
    OutputField6((Close / ma20 - 1) * 100,     "離MA20%");
end;

三週期框架（CIO 操作法）
月線定多空（大方向：多頭才做多）→ 週線定趨勢（波段慣性：底底高/均線多排）→ 日線定買點（時機：拉回均線的反彈+量能確認）。三層全綠才進場 = 順大勢、擇時機，防「日線漂亮但月線空頭」的接刀。
四個變體定位
Ⓐ 底部守穩·量縮轉增=寬鬆版每日監控（Lv0~3 分級）｜Ⓑ 月MACD金叉起漲=金叉當月才觸發（稀有·強訊號）｜Ⓒ 多頭拉回買點=操作主打（含進出場價/風報比/部位建議）｜Ⓓ 站上均線拉回紅K=無指標對照組（K棒+量+結構+均線）。同一池同一天四版並排 → 互為對照。
