📜 XS 選股腳本原始碼 — 多頭拉回買點（含進出場價）（XQ 可直接複製 — 點開）
// =====================================================================
// 多頭拉回買點 選股（3P Ⓒ 含進出場價·做多版; 原名 三層架構操作策略）
//   月：定多空  → 月收盤 >= 月MA12 (年線) 且 月MACD DIF > 0
//   週：定趨勢  → 4 項取 3：站上週MA20 / 多頭排列 / 底底高 / 週MACD金叉中
//   日：定買點  → 拉回 MA20 反彈 + 量增（首選做多訊號）
// 同步輸出：成交金額(億)、進場價、停損價、目標價、風報比、週評分
// =====================================================================
SetTotalBar(2000);

// === 變數宣告 ===
// 月線 MACD
var: monthDif(0), monthMacd(0), monthOsc(0);
// 週線 MACD
var: weekDif(0), weekMacd(0), weekOsc(0);
// 均線
var: monthMA12(0);
var: weekMA5(0), weekMA10(0), weekMA20(0);
var: ma20(0);
// 週波段低
var: wL1(0), wL2(0);
// 三層條件旗標
var: condM(false), condW(false), condD(false);
// 週線分項得分（4 項）
var: scoreW(0);
// 進出場
var: entryPrice(0), stopPrice(0), targetPrice(0), rrRatio(0);

// =====================================================================
// 計算所有技術指標
// =====================================================================
MACD(GetField("收盤價","M"), 12, 26, 9, monthDif, monthMacd, monthOsc);
MACD(GetField("收盤價","W"), 12, 26, 9, weekDif, weekMacd, weekOsc);

monthMA12 = Average(GetField("收盤價","M"), 12);
weekMA5   = Average(GetField("收盤價","W"), 5);
weekMA10  = Average(GetField("收盤價","W"), 10);
weekMA20  = Average(GetField("收盤價","W"), 20);
ma20      = Average(Close, 20);

wL1 = Lowest(GetField("最低價","W"), 8);
wL2 = Lowest(GetField("最低價","W")[8], 8);

// =====================================================================
// 第一層：月線定多空
// =====================================================================
condM = (Close >= monthMA12) and (monthDif > 0);

// =====================================================================
// 第二層：週線定趨勢（4 項至少達成 3 項）
// =====================================================================
scoreW = 0;
if Close >= weekMA20 then scoreW = scoreW + 1;                       // 站上週MA20
if (weekMA5 > weekMA10) and (weekMA10 > weekMA20) then scoreW = scoreW + 1;  // 多頭排列
if wL1 > wL2 then scoreW = scoreW + 1;                                // 底底高
if weekDif > weekMacd then scoreW = scoreW + 1;                       // 週MACD金叉中
condW = scoreW >= 3;

// =====================================================================
// 第三層：日線定買點（拉回 MA20 反彈式）
//   (a) 今日收盤 ≥ MA20（站上 MA20，避免抓下跌中的破線）
//   (b) 近 3 日內最低有觸碰 MA20（容差 1%）
//   (c) 今日收盤 > 昨日收盤（反彈）
//   (d) 今日量 > 5 日均量 × 1.2（量增確認）
// =====================================================================
condD = (Close >= ma20)
    and (Lowest(Low, 3) <= ma20 * 1.01)
    and (Close > Close[1])
    and (Volume > Average(Volume, 5) * 1.2);

// =====================================================================
// 進出場價格計算
//   進場價 = 當前收盤
//   停損價 = max( MA20 × 0.99 , 進場 × 0.93 )  ← 取較近者（損失較小）
//   目標價 = 進場 × 1.30  ← 預設目標 +30%
//   風報比 = (目標 - 進場) / (進場 - 停損)
// =====================================================================
entryPrice  = Close;
stopPrice   = ma20 * 0.99;
if (Close * 0.93) > stopPrice then stopPrice = Close * 0.93;
targetPrice = Close * 1.30;
if (Close - stopPrice) <> 0
    then rrRatio = (targetPrice - Close) / (Close - stopPrice)
    else rrRatio = 0;

// =====================================================================
// 共振觸發：三層全綠才回傳
// =====================================================================
if condM and condW and condD then begin
    ret = 1;
    // order:=1 → 結果頁依成交金額由大到小排序
    OutputField1(GetField("成交金額(億)","D"), "成交金額(億)", order:=1);
    OutputField2(entryPrice,  "進場價");
    OutputField3(stopPrice,   "停損價");
    OutputField4(targetPrice, "目標價");
    OutputField5(rrRatio,     "風報比");
    OutputField6(scoreW,      "週評分");
end;

三週期框架（CIO 操作法）
月線定多空（大方向：多頭才做多）→ 週線定趨勢（波段慣性：底底高/均線多排）→ 日線定買點（時機：拉回均線的反彈+量能確認）。三層全綠才進場 = 順大勢、擇時機，防「日線漂亮但月線空頭」的接刀。
四個變體定位
Ⓐ 底部守穩·量縮轉增=寬鬆版每日監控（Lv0~3 分級）｜Ⓑ 月MACD金叉起漲=金叉當月才觸發（稀有·強訊號）｜Ⓒ 多頭拉回買點=操作主打（含進出場價/風報比/部位建議）｜Ⓓ 站上均線拉回紅K=無指標對照組（K棒+量+結構+均線）。同一池同一天四版並排 → 互為對照。
資料 / 池 / 更新
池 = 中型100 + 權值100 去重 150 檔（舊 3Pstrategy 系統快照·data/agents/p3_strategy/pool.json，可編輯）；本次有效 150 檔。
價量 = 本機價格庫 load_price 日線（FinMind 還原價，R6 只讀）→ pandas resample 週(W-FRI)/月；量單位 = 股（成交金額 = 收盤×量÷1e8 億）。
每日由 orchestrator 排程自動重建（22:00 資料更新後）。
與舊系統（YOSHI 3Pstrategy）的差異
① 舊版 yfinance 原始價 → 本版 FinMind 還原價：所有條件皆為比值/均線口徑，判定結果一致，唯絕對價位（進場/停損顯示值）在除權息前後略異。② 舊版資料經 Excel 中間檔 → 本版直讀價格庫，無中間檔。③ 舊機 15 排程 2026-07-09 全退役，本頁為重建後的唯一維護版。
