📜 XS 選股腳本原始碼 — 站上均線拉回紅K（純價量）（XQ 可直接複製 — 點開）
// =====================================================================
// 站上均線拉回紅K 選股（3P Ⓓ 純價量版·做多; 原名 三層架構 裸K版）
//   月：1 條 → 月收盤 >= 月MA12
//   週：2 條 → 站上週MA20 + 底底高（近 8 週低 > 前 8 週低）
//   日：4 條 → 紅K + 拉回MA20 + 站回MA20 + 量增 1.2x
// 進場/停損(-5%)/目標(+20%)/風報比 一起輸出。依成交金額由大到小排序。
// =====================================================================
SetTotalBar(2000);

// === 變數宣告 ===
var: monthMA12(0);
var: weekMA20(0);
var: wL1(0), wL2(0);
var: ma20(0), avgVol5(0);
var: condM(false), condW(false), condD(false);
var: entryPrice(0), stopPrice(0), targetPrice(0), rrRatio(0);

// === 計算 ===
monthMA12 = Average(GetField("收盤價","M"), 12);
weekMA20  = Average(GetField("收盤價","W"), 20);
wL1 = Lowest(GetField("最低價","W"), 8);
wL2 = Lowest(GetField("最低價","W")[8], 8);
ma20    = Average(Close, 20);
avgVol5 = Average(Volume, 5);

// =====================================================================
// 第一層 月線定多空：1 條
// =====================================================================
condM = Close >= monthMA12;

// =====================================================================
// 第二層 週線定趨勢：2 條全要
//   (a) 站上週MA20
//   (b) 底底高（近 8 週低 > 前 8 週低）
// =====================================================================
condW = (Close >= weekMA20) and (wL1 > wL2);

// =====================================================================
// 第三層 日線定買點：4 條全要
//   (a) 今日紅 K（收 > 開）
//   (b) 近 5 日內有拉回 MA20（容差 1%）
//   (c) 今日收盤站回 MA20
//   (d) 量增（今日量 > 5 日均量 × 1.2）
// =====================================================================
condD = (Close > Open)
    and (Lowest(Low, 5) <= ma20 * 1.01)
    and (Close >= ma20)
    and (Volume > avgVol5 * 1.2);

// =====================================================================
// 進場 / 停損 / 目標
//   進場 = 收盤
//   停損 = max( MA20 × 0.99, 進場 × 0.95 )  ← 取較高（損失較小）
//   目標 = 進場 × 1.20
// =====================================================================
entryPrice  = Close;
stopPrice   = ma20 * 0.99;
if (Close * 0.95) > stopPrice then stopPrice = Close * 0.95;
targetPrice = Close * 1.20;
if (Close - stopPrice) <> 0
    then rrRatio = (targetPrice - Close) / (Close - stopPrice)
    else rrRatio = 0;

// =====================================================================
// 三層全綠才觸發
// =====================================================================
if condM and condW and condD then begin
    ret = 1;
    OutputField1(GetField("成交金額(億)","D"), "成交金額(億)", order:=1);
    OutputField2(entryPrice,  "進場價");
    OutputField3(stopPrice,   "停損價");
    OutputField4(targetPrice, "目標價");
    OutputField5(rrRatio,     "風報比");
end;

三週期框架（CIO 操作法）
月線定多空（大方向：多頭才做多）→ 週線定趨勢（波段慣性：底底高/均線多排）→ 日線定買點（時機：拉回均線的反彈+量能確認）。三層全綠才進場 = 順大勢、擇時機，防「日線漂亮但月線空頭」的接刀。
四個變體定位
Ⓐ 底部守穩·量縮轉增=寬鬆版每日監控（Lv0~3 分級）｜Ⓑ 月MACD金叉起漲=金叉當月才觸發（稀有·強訊號）｜Ⓒ 多頭拉回買點=操作主打（含進出場價/風報比/部位建議）｜Ⓓ 站上均線拉回紅K=無指標對照組（K棒+量+結構+均線）。同一池同一天四版並排 → 互為對照。
資料 / 池 / 更新
池 = 中型100 + 權值100 去重 150 檔（舊 3Pstrategy 系統快照·data/agents/p3_strategy/pool.json，可編輯）；本次有效 150 檔。
價量 = 本機價格庫 load_price 日線（FinMind 還原價，R6 只讀）→ pandas resample 週(W-FRI)/月；量單位 = 股（成交金額 = 收盤×量÷1e8 億）。
市場別＋所屬類股 = sector_map.json（XQ 產業對照表：上市/上櫃·官方分類＋XQ 細產業）。
每日由 orchestrator 排程自動重建（22:00 資料更新後）。
