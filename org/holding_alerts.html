🧠 監控層怎麼運作（閉環）
買→持→監控→賣 的迴圈:
研究部(各 score/警示 agent → FS) → 經理讀 FS bid 選股 → decision 持倉 → 本頁(監控層)把每位經理的持股 ⋈ 各警示來源 → 統一燈號 → 寫回 FS → 經理下一期競標時，對亮紅燈的持股自動降 bid → decision 減碼/換出。

單一調校點:嚴重度模型、動作對應、扣 bid 全在 holding_alerts agent，以後只要優化這一支即可。
燈號需多訊號共振:單一輕警示(如僅結構劣化)不足以亮燈;兩個以上來源同時警示才轉黃/紅(避免單一指標誤殺)。
