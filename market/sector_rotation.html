定義
RS = 族群指數 ÷ 大盤指數(相對強度)　RS Ratio = RS(t) ÷ SMA₁₄(RS) × 100(X 軸;>100 = 相對強)
RS 動量 = (RS(t) − RS(t−5)) ÷ RS(t−5) × 100（Y 軸;>0 = 上升中）
象限:RS Ratio>100 且 動量>0 → 領跑;>100 且 <0 → 轉弱;<100 且 <0 → 落後;<100 且 >0 → 轉強

🚀 動力學分析 (Kinematics) — Ṙ(t) + R̈(t) + L
把每族群當 2D 空間質點：R(t) = ( rs_ratio(t), rs_momentum(t) )
速度 Ṙ(t) = dR/dt → 速率 |Ṙ| + 方向 ∠Ṙ
加速度 R̈(t) = d²R/dt² → 切向 a_t（加速/減速）+ 法向 a_n（轉彎強度）+ 曲率 κ
能量率 Ė = Ṙ · R̈ → 動能擴張/衰退
角動量 L = (R - 中心) × Ṙ → 角速度 ω = L / |R-中心|²，>0 = CCW 順向；<0 = CW 反向
預測 R(t+5) ≈ R + 5Ṙ + 12.5R̈ → 5 日後落在哪個象限
⚠ 用 90 日資料 + Z-score 標準化 + Gaussian 平滑後算二階導數，抑制噪音
