⏱ 範圍
🧠 DS 五層權重 + 三方投票 + 派發的兩種臉
DS = 0.30×價量 + 0.25×籌碼集中↓ + 0.20×Box劣化 + 0.15×法人轉弱 + 0.10×融資升溫

① 價量背離(Effort vs Result):同時認「爆量滯漲」(量大漲不動) 與「量縮無力」(價漲量縮、在高檔)。南電 2021/11 頂部就是量縮型，傳統「爆量=出貨」會漏。
② 籌碼集中↓:主力×賣買家差 (NBC15_Z60↓ + 集中度 SBD_Z60↓ + CMG 轉 Q3/Q4)。
③ Box 劣化:ΔSH 創高幅度縮 + 回檔加深 (box_structure degradation)。
④ 法人轉弱:投信/外資由買轉賣。
⑤ 融資升溫:融資快增 × 價創高 = 散戶接手 (最危險)。

三方投票:不讓單一 Agent 判出貨 —— Box(結構變差) + CSA(籌碼擴散) + Wyckoff(派發相位) 各投一票。DS 分數常在三票確認之前就抬升，正是「邊拉邊出」要的早期窗口 (主力跑了、散戶還在衝)。
📚 主力「邊拉邊出」— 完整看法
本質:價格仍創高，但籌碼由強手逐漸轉移到追價者，很接近 Richard Wyckoff 的派發 (Distribution)。重點不是「有人賣」，而是上漲效率變差 + 籌碼結構惡化。很多主升段也會爆量，關鍵是：量放大後，趨勢結構有沒有繼續改善。
1. 價量結構（最重要）
① 價漲量增、但漲幅縮小（Effort vs Result 背離）
早期主升 量↑↑ → 價↑↑；派發期 量↑↑ → 價只小漲。代表買盤很熱、但大量賣壓被吸收（上方有人供貨）= 典型「出貨需要流動性」。
階段成交量漲幅解讀
早期主升2 萬張+7%量增價漲, 效率高
派發期8 萬張+2%買盤很熱、但大量賣壓被吸收=上方有人供貨
② 爆大量長上影:盤中拉高 → 散戶/FOMO 追進 → 尾盤大單倒貨壓回。波段漲幅已大、遠離 MA20、媒體開始熱 時更需注意。
③ 價創新高、但量能不再有效推升:邊際推升效率下降。PE = 漲幅 / 成交量增幅持續下降 → 派發機率提高。（本系統採 20 日滾動窗，並排除漲停鎖死的惜售低量，避免假性訊號。）
2. 技術結構（Box Agent）
不看單根 K，看 Box 劣化:健康 N 形 = 頭頭高 / 底底高 / Box 上移；邊拉邊出 = 創高有限（ΔSH = SHn − SHn−1 下降）+ 回檔加深（Pullback = SH/(SH−SL) 上升）= 拉升力量下降、但賣壓增加。
3. 籌碼現象（CSA / 主力×賣買家差）
① NBC15 下降:前 15 大券商買賣差下降甚至翻負（價創高但籌碼背離）。
② 籌碼集中度下降:價格創高，但集中度 Z 由 +2σ → +0.5σ → −1σ =「價格還有人推，但貨正在散」。
③ CMG（籌碼 RRG）:健康在 Q1（強籌碼累積）；邊拉邊出 Q4 → Q3（價格還強、但籌碼動能已轉弱）。
4. 法人結構
初期:價↑ + 法人↑ + 融資↑少 = 健康；後期:價↑ + 投信↓ + 外資↓ + 融資↑↑ = 籌碼交換。尤其股價創高 + 融資快速增加，通常不是好訊號（散戶接手）。
5. Distribution Score（本頁 DS）
DS = 30% 價量背離 + 25% 籌碼集中下降 + 20% Box劣化 + 15% 法人轉弱 + 10% 融資升溫
分級:0-30 健康拉升 / 30-60 邊拉邊調節 / 60-80 高度派發 / >80 波段危險。
不讓單一 Agent 判「出貨」 → Box（結構變差）+ CSA（籌碼擴散）+ Wyckoff（派發相位）三者投票。這比傳統「爆量=出貨」可靠很多，因為很多主升段也會爆量 —— 關鍵始終是:量放大後，趨勢結構有沒有繼續改善。
