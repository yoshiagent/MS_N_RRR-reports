④ 四象限交叉（階梯 × 我的價量籌碼訊號）
A 最佳進攻：注意1-2次 + 主力進貨 + accumulation + CMG升 + Wyckoff D/E + 箱型突破 + 融資未爆 → 異常價量剛被官方標記、主力與籌碼集中同步 → 回測箱頂/均線不破分批進，不追連續漲停。
B 可續抱不追：注意3次+ + 續創高 + CMG維持 + 主力未賣 + 融資溫和 → 主升中後段 → 續抱，用前低/5日線/箱頂移動停利。
C 處置觀察：處置第1次 + 量縮價穩 + 籌碼不散 + 法人/主力仍買 → 流動性降但供給未釋出 → 有部位可續抱；新進等解除後確認。
D 高風險出貨：處置 + distribution + 融資爆增 + CMG降 + topping_risk + 分點/法人賣超 → 熱度最高時籌碼從強手轉弱手 → 不新進、減碼/出場；解除後破處置低點不再期待洗盤。
⑤ 兩個關鍵刻度怎麼讀
注意累計次數：1次=雷達名單(不買、開始觀察)；2-3次=甜蜜點(需主力進貨+CMG升+融資未爆)；4-5次=高熱(勝率下降,停利停損優先)；6次+或接近官方處置門檻(近10日6次/近30日12次)=危險區(問題不是會不會再漲,而是被反殺時出不出得掉)。甜蜜點不是固定第幾次,而是「位置」——同樣第2次,剛突破箱型 vs 已漲80%,意義天差。

處置分盤等級：5分盤=第一次降溫；20分盤或第2次以上處置=更嚴重的熱度與流動性壓縮。是頂還是洗盤,看三件事：① 處置前是連續漲停爆量散戶融資追入(→頂機率高)還是溫和；② 處置中量縮價不破、CMG不散、主力沒賣(→偏洗盤/鎖籌)；③ 解除後 1-3 日能創高(強) vs 放量不漲/破低(出貨確認)。

⑧ 陷阱 / 反指標（避免一廂情願）
① 把注意當利多——注意只是「異常」，不是買進認證。
② 把處置當洗盤——有些是洗盤，但更多是流動性變差後的出貨場。
③ 忽略融資——融資暴增=散戶接力，反而讓主力更好出貨。
④ 只看次數不看位置——同第2次注意，剛突破 vs 漲80%後，天差地別。
⑤ 解除後追高——解除若補量不漲，常是最後的流動性出口。
⑥ 倖存者偏誤——你會記得少數處置後續飆的，卻忽略大量回落的樣本。

丟給 gemini(3.5 Flash) 與 codex(GPT-5.5) 兩位顧問獨立批判，兩位都肯定定位正確，但不約而同指出同樣的風險(獨立撞同點＝最可信):
共識批評（gemini 與 codex 獨立撞同點）結論
PRE 段倖存者偏誤：事後把被處置股往前推 20 天標記＝look-ahead bias✅ 必須有對照組（同樣強但沒被處置）
ACTIVE 段指標失效：分盤+預收款券→CSA是人造集中、RS虛胖，會被對倒騙✅ 要加 Liquidity Impairment（交易摩擦）特徵
Volume Pressure 不穩：量變動趨零→極端離群值；制度扭曲✅ VP 降為候選特徵，別當核心
Lock Score 風險：可能只是 CSA/Box/RS 既有因子再包裝✅ 先證明有『增量資訊』再定 40/30/20/10 權重
ML Fingerprint：處置樣本太少→過擬合✅ 先存資料、暫不上 ML
MVP 第一步✅ 先做對照組 event study（=存亡測試）
🔑 兩位都要補的「缺失特徵」（CIO 原設計沒有的）
1. Liquidity Impairment Score（處置造成的可交易性折損）：分盤等級/撮合頻率、漲跌停鎖死比例、有效成交 vs raw volume。沒這層，ACTIVE 段量價訊號都可能是制度扭曲而非真籌碼。
2. 資券/現償壓力：券資比、融資維持率、現股當沖比、強制回補（gemini:比 CSA 更能定斷頭暴跌或軋空）。
3. 交易所規則預測器：規則是公開公式 → 直接算「再漲一檔就進注意/處置」的精確機率，別猜 PRE。
⚖️ codex 的存亡閘門：「DRA 必須證明在既有 CSA/Box/Wyckoff/RS 之外，處置事件能提供可量化的 regime-change 資訊。證明不了，它就只該是 Discipline 的一個事件標籤，而不是獨立 Agent。」

② 三態分類器（砍掉 WARNING 模糊垃圾桶）:COOLING_CONTINUATION / DISTRIBUTION_RISK / UNDECIDED + 信心分（=CIO「強勢冷卻 vs 出貨派發」核心）
③ Liquidity Impairment 簡版 + 可解釋 Lock Score（權重先不寫死）

⑫ DRA Leverage / Short Layer — v1 封版規格（CIO）
核心原則：資券資料本身不判斷多空，而是判斷「籌碼壓力方向」，必須與 CSA + Box 結合解讀。所以 DRA 不問「融券增加好不好？」，而問：「增加的空方壓力，有沒有被強籌碼吸收？」

架構:DRA Leverage Layer ＝ Margin Engine（融資） + Short Pressure Engine（融券 / 券資比 / 借券餘額 / 借券賣出）。其中借券賣出 = 行動（最高權重），借券餘額只是準備。
槓桿/放空指標🟢 正向（吸收/冷卻）🔴 負向（派發/有效放空）
融資 Margin（散戶槓桿多方）價穩 + CSA Q1 + 融資↓ ＝ 散戶洗出＋主力鎖碼 → Cooling價跌 + CSA Q3 + 融資↑ ＝ 散戶承接 → Distribution
融券 Short（短線空方）融券↑ + 價不跌 + CSA Q1 ＝ 空方燃料↑ → Squeeze 潛力融券↓ + 融資↑ + CSA 弱 ＝ 空方回補完、多方散戶化
券資比 Short/Margin（比單看融券好）融資↓ + 融券↑ + 價穩 + CSA Q1 ＝ 最佳股價跌 + 融券↑ + Box破 ＝ 空方打成功
借券餘額 SBL Bal（法人級空方準備）借券↑ + 價不跌 + CSA集中 ＝ 供給被吸收借券↑ + 價跌 + CSA擴散 ＝ 空方有效
借券賣出 SBL Sell（行動·最高權重）賣出壓力↑ 但 Box守 + CSA Q1 ＝ 強吸收借券賣出↑ + Box破 + CSA Q3 ＝ 派發確認
兩個核心 Score（v1 不混成一個）
A. Cooling / Squeeze Score（處置是否只是洗籌、甚至累積第二波能量？，100制）
＝ 30% CSA Lock + 20% Box Hold + 15% 融資下降 + 15% 券資比改善 + 10% 空方壓力吸收（Absorption ＝ 借券賣出↑ 但 價跌<5% 且 Box未破）+ 10% RS保持

B. Distribution Risk Score（主力是否趁處置派發？，100制）
＝ 30% CSA Distribution + 25% Box Damage + 15% 融資增加 + 15% 借券賣出有效 + 10% RS下降 + 5% 融券下降
情況（CSA × Box × 槓桿）DRA 判斷
CSA Q1 + Box守 + 融資↓ + 券資比↑⭐⭐ Strong Cooling
CSA Q1 + Box守 + 借券賣出↑⭐⭐ 吸收籌碼
CSA Q1 + 融資↑觀察
CSA Q3 + 融資↑派發警告
CSA Q3 + Box破 + 借券賣出↑🔴 Distribution
