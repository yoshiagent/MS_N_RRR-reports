💡 Wyckoff Distribution 核心邏輯 — 5 維派發分數 + UTAD 假突破偵測

🌋 Wyckoff Distribution Phase = 主力在高檔默默出貨給散戶的階段（Accumulation 的鏡像）。本 Agent 對 主流 + 強勢 + 自選 已上漲股池算 5 維派發分數：

① 量價見頂（D1：ATR 放大 + 高檔橫盤 + UTAD 假突破）
② 結構轉弱（D2：n_shape 切到 unhealthy family）
③ 業績轉差（D3：營收/EPS YoY 由正轉負）
④ 大戶出貨（D4：chip_score 低 + 外資連賣 + 融資增）
⑤ 跌破準備（D5：距箱底 近 + 放量 panic）

換股決策：DS 越高 → 越該換出；持有股 DS < 40 可延遲換股。右側 AS 欄位是同股的 Accumulation Score — 矛盾訊號（DS 高 + AS 高）通常代表橫盤拉鋸，應觀察更久。

🌋 UTAD（Upthrust After Distribution）= 假突破後反轉，Distribution Phase C 經典訊號；名稱旁紅徽章即此判定。
