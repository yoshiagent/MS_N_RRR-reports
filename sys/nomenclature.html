🥇 名次與分數排名類指標，注意「名次」與「名次分數」方向相反
名次分數rank_scorehotflow
定義把『名次』翻轉成『分數』，方便做加減運算與斜率計算
公式if amount_rank ≤ 100: rank_score = 101 - amount_rank; else: 0
方向★ 數字越大越好（與『名次』方向相反）
範例amount_rank=1 → rank_score=100；amount_rank=100 → rank_score=1；amount_rank=200 → rank_score=0
名次推進動能（1日）rank_momentumhotflow
定義1 日的名次分數變化（用名次分數差非名次差，方向才正確）
公式rank_momentum = 今日 rank_score − 昨日 rank_score
方向正值 = 表現變好；子卡顯示為『百分位排名』(PR 0-100)
範例rank_score 由 50 → 75 → rank_momentum = +25
名次推進速度（近 5 日）rank_slope_5dhotflow
定義近 5 日名次分數的 OLS 線性回歸斜率
公式slope = Σ(-2·y[-4] -1·y[-3] +0·y[-2] +1·y[-1] +2·y[0]) / 10
方向斜率正 = 名次往前進（表現變好）；子卡顯示為 PR 0-100
範例近 5 日 rank_score = [10,20,30,40,50] → slope = +10/日
名次變動（1日）rank_changehotflow
定義1 日的名次進退步
公式rank_change = 昨日 amount_rank − 今日 amount_rank
方向正值 = 名次往前進（表現變好）
範例昨日 #50、今日 #30 → rank_change = +20（進步 20 名）
成交金額名次amount_rankhotflow
定義當日成交金額在全市場 ~1,935 檔中的排名
公式ROW_NUMBER() OVER (PARTITION BY date ORDER BY close × volume DESC)
方向★ 數字越小越好（#1 = 全市場第 1 名）
範例2330 amount_rank=3 → 當日成交額排第 3 名
成交金額百分位amount_rank_pcthotflow
定義在全市場成交金額排序中的相對位置（0-1）
公式PERCENT_RANK() OVER (PARTITION BY date ORDER BY amount)
方向越接近 1 = 成交額越大
範例amount_rank_pct=0.99 → 排在最後 1%（成交額最大群）
近 20 日上榜天數hot_persistence_20dhotflow
定義近 20 個交易日中『名次進入前 100』的天數
公式Σ(amount_rank[t] ≤ 100 for t in last 20 days)
方向越大 = 持續被資金關注；範圍 0-20
範例近 20 日有 15 天進入前 100 → hot_persistence_20d = 15
💰 量能成交金額相關指標
成交金額amount全系統
定義當日股票交易的金額
公式amount = close × volume
範例收盤 1535 元 × 量 2,148,000 股 → 32.97 億元
量能擴張倍數（vs 60 日均）amount_expansionhotflow
定義當日成交金額相對過去 60 日均的倍數
公式amount_expansion = 當日 amount / 過去 60 日 amount 均值
方向> 1 = 放大；> 2 = 明顯放大；> 3 = 異常爆量
範例今日 30 億，60 日均 15 億 → amount_expansion = 2.0（放大 2 倍）
量能爆發程度（60 日 Z 分數）amount_z_60dhotflow
定義當日成交金額相對 60 日歷史的標準化分數
公式z = (今日 amount − 60 日均) / 60 日標準差
方向z > 2 = 明顯爆量；z > 3 = 極端爆量
範例60 日均 10 億、標準差 3 億，今日 19 億 → z = (19-10)/3 = +3.0
📊 相對強弱個股相對大盤的表現
相對大盤強弱rshotflow
定義個股相對於加權指數（TAIEX）的強弱
公式rs = 個股收盤 / 加權指數收盤
方向rs 上升 = 個股表現優於大盤；下降 = 弱於大盤
範例個股 100 元、TAIEX 17000 → rs = 0.00588；明日個股 110 元、TAIEX 17000 → rs = 0.00647（變強）
相對大盤綜合分rs_combinedhotflow
定義綜合『rs_trend 是否向上』與『近 20 日超額報酬』
公式rs_combined = 50% × rs_trend×100 + 50% × PR(rs_strength_20d)
方向越高 = 相對大盤越強；子卡顯示為 PR 0-100
範例rs_trend=1（+50）+ rs_strength_20d 在前 20%（PR=80，計 +40）→ rs_combined = 90
相對強弱 20 日均線rs_ma20hotflow
定義rs 的 20 日移動平均
公式rs_ma20 = mean(rs, window=20)
方向rs > rs_ma20 = 短期趨勢偏多
相對強弱趨勢rs_trendhotflow
定義rs_ma20 的方向（向上 = 1、向下/持平 = 0）
公式rs_trend = 1 if rs_ma20[t] > rs_ma20[t-1] else 0
方向1 = 多頭趨勢；0 = 空頭趨勢
範例rs_ma20 連續上升 → rs_trend = 1
近 20 日相對大盤超額報酬rs_strength_20dhotflow
定義個股近 20 日報酬減大盤近 20 日報酬
公式rs_strength_20d = ret_20d(個股) − ret_20d(TAIEX)
方向正值 = 跑贏大盤；負值 = 跑輸大盤
範例個股 +20%、大盤 +10% → rs_strength_20d = +10%

🔁 多時段箱型狀態9 + 3 + 1 狀態系統與 healthy 判定（N 形為 above_up 別稱）

範例2330 daily=above_up + [breakout_pending] → 剛突破 1-2 根，最高警覺期

公式True 當 state ∈ {above_up, above_down, below_recovery, inbox_up}
方向★ True = 健康（可進場）/ False = 不健康（降級觀察）
範例2330 healthy=True → 維持「佈局」階段；上述外狀態 → 強制降為「觀察」

定義依「位置 × 方向」二維分類的 9 主狀態 + 3 修飾詞 + 1 unknown。位置由 last_close 與最近 confirmed SH/SL 比較決定（above/in/below）；方向由近 3 根 close 斜率決定（up/down/flat）。N 形 = 本系統的特殊狀態（above_up + breakout_pending = 標準 N 形展開）；本框架涵蓋 N 形 + 破壞 + 反轉 + 盤整等完整生命週期。
公式9 主狀態 keys: above_up / above_down / above_fake / inbox_up / inbox_down / inbox_side / below_up / below_down / below_recovery；3 修飾詞: breakout_pending / breakdown_pending / compression
方向★ 健康族（healthy）= above_up / above_down / below_recovery / inbox_up
範例2330 台積電 週線 = above_up（箱上推進）+ breakout_pending（突破確認中）

公式healthy_pair = A_healthy AND B_healthy；pair_health ∈ {both_healthy, a_only, b_only, none}
方向★ True = 兩股皆 healthy 族，可優先採信換股訊號
範例RR ['2330','2454'] → A_state=above_up, B_state=above_up → healthy_pair=True

60 日 Z 分數z60RRR / Dynamic RSR

公式z60 = (rs_ratio − 60日均) / 60日標準差

RS 比率rr_ratioRSR / RRR / Dynamic RSR
定義兩檔配對股的對數比率（Relative Strength Ratio）
公式rs_ratio = ln(A 收盤 / B 收盤)
方向上升 = A 強於 B；下降 = B 強於 A
範例A=200、B=100 → rs_ratio = ln(2) ≈ 0.693
折返 Z 門檻pullback_zDynamic RSR
定義Z 從極值折返 ≥ Pullback_Z 才換第 2 段（40%）
公式依 vol_ratio 動態決定：低 0.3 / 正常 0.4 / 高 0.5 / 極端 0.7
範例Z 達 +2.5 後折返至 +2.0（差 0.5）→ 若 Pullback_Z=0.4 則觸發
波動比率vol_ratioDynamic RSR
定義短期 vs 長期波動度的比值
公式vol_ratio = STD20(rs_ratio) / STD60(rs_ratio)
方向< 0.8 低波動 / 0.8-1.2 正常 / 1.2-1.6 高 / ≥ 1.6 極端
範例vol_ratio = 1.3 → 高波動配對，動態 RSR 用 Entry_Z = 2.5

公式z = (x − mean) / std
方向|z| > 2 = 統計顯著；|z| > 3 = 極端
範例amount_z_60d = +2.5 → 今日量是 60 日均值的 2.5 倍標準差以上
極端值修剪（winsorize）winsorizehotflow
定義把上下 1% 的極端值用 1% / 99% 分位數截斷
公式x.clip(lower=x.quantile(0.01), upper=x.quantile(0.99))
方向避免極端 outliers 主導 PR 排名
百分位排名（PR）PR / percentile_rankhotflow
定義把任意數值轉成 0-100 的相對排名
公式PR(x) = rank(x, pct=True) × 100；NaN 填 50
方向0 = 末名、50 = 中位、100 = 第 1 名
範例100 檔股票 hotflow_score=71.9 排第 3 → PR ≈ 98
