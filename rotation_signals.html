🔄 換股訊號邏輯說明

🔄 Rotation Signal = Distribution Score 對撞 Accumulation Score 的「換股決策層」。

邏輯：手上派發中（DS 高）→ 候選池吸籌中（AS 高）→ 計算 swap_urgency = max(0, DS-40) + max(0, AS-40) + 同族群 +10。

使用：手上 DS < 40 → 不急（延遲換股，繼續持有）；DS ≥ 60 + 候選 AS ≥ 60 → 強烈換股建議。

🎯 同族群配對 = 族群輪動觀（內部資金移動）；🌋 持有股 UTAD = Distribution Phase C 訊號；🌱 候選股 Spring = Accumulation Phase C 訊號。
