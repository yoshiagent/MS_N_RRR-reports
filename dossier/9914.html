定義
三時段(日/週/月)各取 SHSL Agent 的 wave_state + 4-flag(HH/HL/LH/LL) + 最後確認 SH/SL → 價位帶位置%
回檔箱(箱頂/箱底/世代/趨勢/位移)= box_structure agent 決定;報告頁只渲染(單一決策者)

判讀（波段位置）
wave_state:PULLBACK 拉回中(橘)· REBOUND 反彈中(紫)· BREAKOUT 突破(綠)· BREAKDOWN 跌破(紅)。4-flag 綠(HH/HL)= 多頭結構、紅(LH/LL)= 空頭結構。價位帶:SL(綠)→ 當前(黃)→ SH(紅),位置% = 在區間內位置(越高越接近壓力)。三時段同步(皆 REBOUND/PULLBACK)= 共振訊號;週反彈+日拉回 = 可能買點。⛔ 週線下降箱(LH+LL)= 紀律規範不得操作、持有應出清。

🗺 這個 block（B6 · 定義 · 判讀 · 資料 · 程式 · 換股 — 點開）
定義
河道_PX(t) = 歷史 PE 第 X 分位 × 當日 TTM EPS(t)　TTM EPS = close ÷ PER（FinMind PER 反推, 免額外財報）
5 層帶:<P10 深紫(極便宜) · P10–P30 藍 · P30–P70 綠(合理) · P70–P90 橘 · >P90 紅(極貴)
公允價 = P50 河道(綠虛線);乖離 = (股價 − P50) ÷ P50
判讀（估值分位）
股價白線穿入紅帶 = 統計上極端偏貴(PE 處歷史 ≥90 分位)、穿入深紫帶 = 深度便宜(≤10 分位);回到綠帶 = 估值合理。乖離 % 正 = 比公允價貴。注意:便宜可能是基本面惡化(EPS 下修使河道整體下移)→ 須搭配成長/籌碼判讀,非單看分位機械進出。
操作
點圖／手機點觸 = 定游標(垂直線, 不隨滑鼠);◀◀◀▶▶▶ 鈕或鍵盤 ←→ 移游標(圖不動);游標當日 股價／公允價(P50)／乖離／估值區 顯示在圖上方資訊列。範圍鈕 近3月／近1年／近3年／全部(右端固定最後一天、預設近1年);桌面雙擊圖 = 重設;手機只單點+長按(已關 拖曳/滾輪/雙指/雙觸)。
